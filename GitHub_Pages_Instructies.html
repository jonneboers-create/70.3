Stap voor stap

1
Maak een gratis GitHub account
Ga naar github.com en klik op Sign up. Kies een gebruikersnaam — die wordt onderdeel van je link, dus kies iets als jonne-race of je eigen naam.
https://github.com/signup
2
Maak een nieuw repository aan
Na inloggen klik je rechtsboven op het + icoon → New repository.

Vul in:
• Repository name: race-day
• Zet op Public (verplicht voor gratis hosting)
• Vink aan: Add a README file

Klik Create repository.
3
Upload alle bestanden
In je nieuwe repository klik je op Add file → Upload files.

Sleep alle zes bestanden tegelijk naar het uploadvenster. Wacht tot alle bestanden zijn geüpload (de grote PDFs kunnen even duren).

Klik onderaan op Commit changes.
💡 Sleep ze allemaal tegelijk — dan hoef je maar één keer op Commit te klikken.
4
Zet GitHub Pages aan
Ga naar Settings (tabblad bovenaan in je repository).

Scroll naar Pages in het linkermenu.

Onder Source: kies Deploy from a branch.
Onder Branch: kies main en map / (root).

Klik Save.
5
Wacht 1-2 minuten — dan is je link klaar
Ververs de Settings → Pages pagina. Je link verschijnt bovenaan als het klaar is:
https://JOUWGEBRUIKERSNAAM.github.io/race-day/703_splits_calculator.html
✅ Stuur deze link via WhatsApp naar de familie. Ze hoeven niets te installeren — gewoon openen in de browser.
